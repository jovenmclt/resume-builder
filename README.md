# Resume Builder

A web application for fresh graduates and job seekers to create, customize, and download professional resumes as PDFs. The platform provides a simple, intuitive interface for users to build resumes quickly without technical knowledge.

## About
The Resume Builder helps users design and generate professional resumes efficiently. Users can input personal information, education, work experience, skills, and other details, then save and download their resume in PDF format. The platform focuses on simplicity, usability, and clean design.

## Features
- Create and customize resumes with personal details, education, experience, and skills
- Choose from multiple pre-designed templates
- Preview resumes before saving
- Generate and download resumes as PDF
- Responsive and mobile-friendly design
- AI chat support

## Technologies Used
- Frontend: Vuejs 3, Bootstrap 5, Inertia.js  
- Backend: Laravel 11 Session
- Other Tools: Composer, NPM
